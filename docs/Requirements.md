Base Application Description
Create a base application with a simple to-do list that allows users to:

• Add new tasks.

• Mark tasks as completed.

• Delete tasks.

The application is built with Ionic and Angular, and uses local storage to save the state of tasks.

Add the ability to categorize tasks. Users should be able to:

• Create, edit, and delete categories.
• Assign a category to each task.
• Filter tasks by category.
